# Tflask: 金融交易回测 Python 库

---

## 简介

Tflask 是一个用于金融交易回测的 Python 库，提供了一系列函数来支持各种交易策略的回测与分析。该库主要功能包括止盈止损条件判断、每日持仓状态计算、机器学习模型训练、获取币安API数据、绘制K线图以及策略评价指标计算等。通过 Tflask，用户可以方便地进行交易策略的回测与优化。
